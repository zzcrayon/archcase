# Codex 使用规则

## 基本要求
- 回答请使用中文。
- 每次修改前，先说明准备修改哪些文件。
- 每次只完成我明确提出的任务，不要主动扩展功能。
- 不要进行大范围重构，除非我明确要求。

## 不要读取的目录
除非我明确要求，请不要读取或扫描以下目录：

- node_modules
- dist
- .git
- public 中的大型图片文件

## 优先关注的文件
本项目主要关注以下文件：

- README.md
- src/App.jsx
- src/styles.css
- src/data
- src/assets

## 修改规则
- 修改前先告诉我你的计划。
- 修改后告诉我改了哪些文件。
- 不要自动修改无关文件。
- 不要自动运行大范围扫描。
- 不要反复读取整个项目。

## 项目背景
这是一个建筑案例管理与作品集展示项目，主要用于展示建筑案例、搜索筛选、评分、数据管理和项目作品集介绍。
目标是让网页更适合作为建筑学学生的个人作品集展示。
